### #.19 ENV Variables

#### 1. Sử dụng .dotenv

```bash
npm i dotenv@16.0.3
```

Tạo file .env ở level root và truy cập 1 biến, chúng ta sử dụng cú pháp: `process.env.VARIABLE_NAME`
